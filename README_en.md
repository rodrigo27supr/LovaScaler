# LovaScaler Pro v2.0

LovaScaler Pro is a Windows desktop application for AI image upscaling. It uses the Real-ESRGAN engine to achieve up to 4x enlargement with professional quality and sharpness.

## What's new in v2.0

* **No Freezing**: Async engine processing keeps the UI fully responsive during heavy tasks.
* **Live Preview**: Real-time thumbnail monitoring of the image currently being processed.
* **Progress Bar**: Visual status tracking for your batch processing queue.
* **Path Persistence**: Automatically remembers your last used source and destination folders.

## Main Features

* **Modern UI**: Dark mode design built with CustomTkinter.
* **Batch Processing**: Automatically detect and scale all images within a selected folder.
* **File Safety**: Intelligent file naming prevents overwriting existing files in the destination folder.
* **Dual AI Modes**: Specific models for real photography and anime/illustrations.

## Requirements & Structure

* **Standard Users (.exe)**: Ensure `LovaScaler.exe` is in the same directory as the `bin/` folder.
* **Developers**: Python 3.10+ required, along with `customtkinter` and `Pillow` libraries.
