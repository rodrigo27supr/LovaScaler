# LovaScaler Pro v2.0

LovaScaler Pro es un programa para Windows que utiliza inteligencia artificial (Real-ESRGAN) para ampliar tus imágenes hasta 4 veces su tamaño original sin que se vean pixeladas ni pierdan nitidez.

## Lo nuevo en esta versión

* **Sin bloqueos**: El motor de la IA trabaja de forma independiente a la ventana para evitar que se congele.
* **Vista previa en vivo**: Muestra una miniatura de la imagen procesándose en tiempo real.
* **Barra de progreso**: Seguimiento visual de las tareas pendientes.
* **Memoria de carpetas**: Recuerda las últimas rutas de origen y destino utilizadas.

## Qué puedes hacer con el programa

* **Interfaz Moderna**: Diseño en modo oscuro con CustomTkinter.
* **Procesamiento automático**: Escala carpetas completas de forma automática.
* **Seguridad de archivos**: Si en el destino ya existe una foto con el mismo nombre, le añade un número al final automáticamente.
* **Modos de IA**: Opción para fotos reales y otra para dibujos/anime.

## Cómo usarlo

* **Si usas el ejecutable (.exe)**: Mantén el archivo `LovaScaler.exe` junto a la carpeta `bin/`.
* **Si eres desarrollador**: Necesitas Python 3.10+, y las librerías `customtkinter` y `Pillow`.
